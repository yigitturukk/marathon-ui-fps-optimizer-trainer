## 🛠️ Installation (Windows 10/11)

### Quick Start
1. Extract the archive to any folder (e.g., `C:\MarathonOptimizer`).
2. **Run `MarathonOptimizer.exe` as Administrator**.
3. Click "Apply UI Fix" and "Optimize FPS".
4. (Optional) Enable trainer features from the "Trainer" tab.

## ❗ Troubleshooting

| Problem | Solution |
|---------|----------|
| Tool won't start | Run as Administrator. Install .NET 4.8+ |
| UI fix not applying | Restart game after applying. Some settings require a reboot. |
| FPS boost not working | Try different preset (Low, Medium, High). Reset to default if needed. |
| Trainer features cause crash | Disable and restart. Update BattlEye? |
| Antivirus false positive | Add `MarathonOptimizer.exe` and `trainer_module.dll` to exclusions – they're safe. |
